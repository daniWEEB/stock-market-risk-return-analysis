# Stock Market Risk & Return Analysis (2018–2023)
Statistical analysis project examining the risk, return, and market sensitivity of major US stocks using daily market data. The project applies descriptive statistics and CAPM concepts to support data-driven investment and risk assessment decisions.
## Objective
- Compare risk and return characteristics of selected stocks
- Analyse volatility and return distributions
- Measure market sensitivity using CAPM beta
## Scope
- Stocks: Microsoft, Apple, Tesla
- Market Benchmark: S&P 500
- Data Frequency: Daily prices
- Time Period: 2018–2023
## Key Analysis
- Price trend analysis
- Daily return calculation
- Descriptive statistics (mean, volatility, min, max)
- Return distribution analysis (histograms)
- Market relationship analysis (scatter plots)
- CAPM beta and R² interpretation
## Tools Used
- Google Sheets
- Statistical Analysis
- Financial Analytics
- Data Visualisation
